# 准备工作

TIP

本文的操作都是基于Windows 7 sp1系统

# 目的

拥有属于自己的网络日志！

# 安装Git

Windows：下载并安装 git在新窗口打开。

# 安装NodeJS

Windows：下载并安装 Node.js在新窗口打开（推荐）或者通过 nvs在新窗口打开 安装。

通过 nvs 安装 Node.js 步骤：

  1. nvs 安装时一路Next直到出现successfully字样，再点Finish按钮完成安装。
  2. Windows 7 SP1 单击左下角的 Windows 图标，在搜索框中键入cmd，单击cmd.exe，弹出的窗口叫终端窗口。终端窗口中执行命令：
nvs add 12.18.2 #=>添加node-v12.18.2
nvs use lts     #=>使用当前版本
nvs link lts    #=>配置为默认版本
